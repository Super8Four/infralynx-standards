Shared governance repository for InfraLynx collaboration standards.

## Contents

- `.github/` for GitHub templates, ownership rules, and repository-facing standards
- `governance/` for branch strategy, branch protection, and default repository settings
- `policies/` for contribution, security, support, and changelog standards
- `templates/` for labels, milestones, project board structure, and usage guidance
